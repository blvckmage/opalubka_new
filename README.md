# Опалубка CRM (демо)

Простая CRM на PHP + SQLite для учёта аренды опалубки.

Как запустить (локально):

1. Убедитесь, что у вас установлен PHP (>=7.4) и включён модуль pdo_sqlite.
2. Запустите встроенный сервер: php -S localhost:8000 -t .
3. Откройте http://localhost:8000/

Система автоматически создаст базу данных в `data/opalubka.sqlite` при первом запуске.

Функции:
- Панель управления с базовой аналитикой
- Создание аренды, список заказов
- Учет склада и движений опалубки
- Список клиентов

Дальше можно добавить: аутентификацию, экспорт, фильтры, графики (Chart.js) и т.д.
